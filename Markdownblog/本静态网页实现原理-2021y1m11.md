# 介绍

> 截至今日网页的语言占比：css 70.3%  html 17.3% javascript 12.4%

## 网页更新的方法

迫于github page的根本限制，我只能在本地写好一个位于Markdown文件夹下的index.html，并在其中加入此文件夹的信息。再在c端请求html解析后得到文件夹下的内容。进而通过js生成blog的列表，并索引到相应的文件。在得到服务器返回的md字符串后，我可以用第三方的解析器（marked等等）将其解析为html字符串。生成html字符串后，请求root目录里的预先设定好的css文件，将其与字符串的形式并结合style标签的scoped属性将css的作用域限制在markdown字符串内。最后，再将css与html以一定结构拼接，并嵌入网页，就完成了。
